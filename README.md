# FamilyEditor

**FamilyEditor** — это плагин для Revit на платформе .NET Framework 4.8, предназначенное для управления файлами семейств (.rfa) и шаблонов проектов (.rte).
Работает в системе MagicEntry

## Возможности

- **Пересохранение:**  
  Позволяет пересохранить файл семейства или шаблона проекта в той же директории сначала под новым именем, а потом под старым.
  Таким образом файл может стать значительно меньше.

- **Открытие директории:**  
  Открывает папку, в которой располагается текущий файл (семейство, шаблон проекта, локальная копия проекта.  
  

## Особенности

- Все сообщения и шаблоны форматирования хранятся централизованно в классе `Messages`.
- Все действия сопровождаются информативными сообщениями для пользователя.

## Требования

- .NET Framework 4.8
